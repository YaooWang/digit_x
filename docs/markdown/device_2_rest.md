# 装置_2_测试
装置2由控制端、受控端和电脑端三部分组成。主要技术包括：语言识别技术应用、分离体信息无线传输控制，以及时间模块、光敏敏传感器和热释红外传感器环境监测，控制装置行为。当，有人经过时，装置自动点亮(控制端为“北斗七星”，受控端为“闪电”LED)，经过30s后会自动熄灭；控制端具有语言识别模块，分别控制受控端的“闪电”LED和一字激光，以及播放数字营造学社简介。指令分别为：
